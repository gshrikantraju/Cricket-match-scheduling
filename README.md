# Cricket-match-scheduling\n
Scheduling cricket match between teams present in list such that each and every team plays only one match with other teams.\n
Example: \n
  teams = ['A', 'B', 'C', 'D']\n
  OUTPUT =  A Vs. B,\n
            A Vs. C,\n
            A Vs. D,\n
            B Vs. C,\n
            B Vs. D,\n
            C Vs. D\n
