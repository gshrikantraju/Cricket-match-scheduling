# Cricket-match-scheduling
Scheduling cricket match between teams present in list such that each and every team plays only one match with other teams.
Example: 
  teams = ['A', 'B', 'C', 'D']
  OUTPUT =  A Vs. B
            A Vs. C
            A Vs. D
            B Vs. C
            B Vs. D
            C Vs. D
